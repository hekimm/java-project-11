# Multiplication Table Generator

A Java console program that prints out a multiplication table from 1 to 10. Each number is multiplied by numbers from 1 to 10, displaying the results in a structured format. This program is a basic introduction to loops and nested loops in Java, making it ideal for those just getting started with programming.

## Project Description

This simple program uses nested loops to generate and print a multiplication table for numbers 1 through 10. It is designed to help beginners understand the concepts of for loops and how they can be used within each other to perform repetitive tasks efficiently.

## Features

- Prints a 10x10 multiplication table.
- Demonstrates the use of nested loops in Java.
- Outputs are displayed in a clear and organized manner.

## How to Run

To run this program, ensure you have Java installed on your computer. Follow these steps:

1. Download the `JavaApplication33.java` file.
2. Open your command line interface.
3. Navigate to the directory where the file is stored.
4. Compile the Java file using the command: `javac JavaApplication33.java`.
5. Run the compiled Java program with: `java JavaApplication33`.
